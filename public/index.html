Dropbox API: Tutorial

Welcome to the exciting word of APIs. With a little practice, APIs will become another handy tool for your web development career.

Specifically, we'll be looking at the Dropbox API with JavaScript. The full documentation is available on GitHub: Dropbox-SDK-js.

In this tutorial, you will learn about:

  • Installing an API that utilizes Node.js
  • Selecting proper authentication based on needs
  • Navigating folders and files in Dropbox

Getting Started

There are many languages to use with Dropbox. This tutorial utilizes Dropbox for Javascript.

Installation

Follow the Dropbox JS documentation to install Dropbox in your program folder with npm.

Because we will be using Node to access Dropbox, we will need a way to bundle our JavaScript files into a single file. This is where Browserify comes into play. Browserify allows us to use require('dropbox').

In the same folder you installed Dropbox, install Browserify with the following command.

npm install browserify --save

Register Your App

Dropbox is a service that requires login. Whether you are wishing to access your own Dropbox files or the user's Dropbox files, you will need to register your website as a new app on Dropbox.

  1. Go to Dropbox Developers to Create app.
  2. Choose the Dropbox API.
  3. Choose the Full Dropbox option.
  4. Name your app.

You can find all your registered apps at My Apps on Dropbox Developers. This is also where you can adjust settings for your app or even delete your app from the Dropbox service.

Authentication

Dropbox authenticates users using OAuth 2 flow. That is, you must ask for access to files from your users. Once approved, an access token is returned, which allows you to access their Full Dropbox.
